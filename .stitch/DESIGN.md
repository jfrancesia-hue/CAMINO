# Design System: Camino

## 1. Visual Theme & Atmosphere

Camino debe sentirse como una **catedral catolica espanola convertida en producto digital**: viva, luminosa, emocional, premium y profundamente humana. La belleza no es decoracion; es parte del acompanamiento. La interfaz debe emocionar antes de explicar.

La direccion visual mezcla:

- **Sagrada Familia:** luz organica, color espiritual, formas no rigidas.
- **Catedrales de Sevilla y Toledo:** oro, piedra, profundidad, solemnidad calida.
- **Vitrales espanoles:** cobalto, turquesa, bermellon, azafran y marfil.
- **Producto moderno:** claridad, accesibilidad, performance, CTA visibles.

Palabras guia:

**Spanish cathedral, vivid stained glass, sacred color, Marian blue, liturgical gold, vermilion glass, turquoise relief, saffron light, mosaic glow, organic arches, editorial luxury, chapel silence, modern Catholic sanctuary, emotional wellness.**

## 2. Color Palette & Roles

### Core

- **Marian Night Blue (#070d1c):** canvas principal, profundidad espiritual, noche contemplativa.
- **Cathedral Blue (#0c2147):** superficies oscuras, overlays, hero y app shell.
- **Marian Blue (#14366f):** identidad catolica, confianza, acentos institucionales.
- **Warm Ivory (#fff8e8):** texto principal, calma, humanidad y fondos de descanso.

### Sacred Vivid Accents

- **Spanish Cobalt (#1556b7):** vitral azul vivo, energia mediterranea, bloques de impacto.
- **Sacred Turquoise (#11a7a7):** alivio, respiracion visual, mosaico, estados positivos.
- **Vermilion Glass (#d94a35):** emocion, calidez, humanidad, contraste dramatico.
- **Saffron Light (#f0b429):** sol espanol, celebracion, highlights y microinteracciones.
- **Liturgical Gold (#d7ad4f):** CTA principal, luz, esperanza, bordes sagrados.
- **Candle Gold (#f6dfa2):** hover, brillos, cruces, badges y textos destacados.

### Supporting

- **Parchment (#eadcc1):** secciones editoriales, contenido espiritual, lectura.
- **Sacramental Wine (#7f2437):** profundidad emocional, alertas suaves, contraste.
- **Olive Quiet (#6f8051):** naturaleza, crecimiento, calma pastoral.
- **Coral Care (#ef7979):** crisis, errores y advertencias humanas.

## 3. Color Rules

- No usar todo azul. Cada pantalla importante necesita al menos dos acentos vivos: oro + turquesa, oro + bermellon, o cobalto + azafran.
- El fondo puede ser oscuro, pero siempre debe haber luz visible: fotografia, halo, vitral, linea dorada o fondo marfil.
- El dorado se reserva para accion, cruz, foco visual o significado espiritual. No usarlo como simple relleno.
- Las alertas de crisis deben conservar legibilidad y cuidado; no deben usar rojo agresivo puro.

## 4. Typography Rules

- **Hero y grandes titulares:** serif editorial (`Georgia`, `Times New Roman`, serif) con escala grande, ritmo solemne y aire.
- **Producto y formularios:** sans serif del sistema, clara y moderna.
- **Badges:** mayusculas, tracking amplio, pequenos y liturgicos.
- **Copy:** frases humanas, breves, sin sermon. La interfaz debe hablar como alguien que acompana, no como cartel institucional.

## 5. Image Direction

Las imagenes son obligatorias en landing, producto publico y momentos emocionales.

Usar imagenes reales de:

- Catedrales espanolas o interiores catolicos luminosos.
- Vitrales con color fuerte.
- Velas, Biblia, bancos de iglesia, detalles de piedra, mosaicos.
- Senderos con amanecer para simbolizar proceso.
- Comunidad real y cercana, no stock corporativo.

Evitar:

- Fondos abstractos como unica imagen.
- Fotos oscuras donde no se reconoce el objeto.
- Cruces diminutas o simbolos ambiguos.
- Imagenes genericas de wellness sin fe visible.

## 6. Logo & Sacred Symbols

- El logo debe leerse en 1 segundo: una **cruz clara** dentro de un medallon luminoso.
- La cruz puede aparecer como detalle recurrente, pero no debe saturar.
- Usar arcos, rosetones, vitral, velas y pergamino como lenguaje secundario.
- Si una card usa icono, debe ir acompanada de imagen, textura o contenido real. No usar "logos vacios".

## 7. Layout Principles

- Primer viewport: marca grande, cruz visible, fotografia real protagonista y CTA claros.
- Composicion editorial: imagen protagonista + texto amplio + paneles superpuestos.
- Evitar grids demasiado simetricos. Usar collage, solapes, columnas irregulares y cards de distinta altura.
- Alternar ritmos:
  - Hero inmersivo.
  - Seccion de dolor con aire y contencion.
  - Seccion marfil tipo editorial.
  - Seccion vitral viva.
  - CTA con foto emocional.
- Mantener suficiente contraste y respiracion: lo vivo no debe volverse ruido.

## 8. Component Styling

### Buttons

- Primario: dorado liturgico con texto oscuro, sombra calida y hover marfil.
- Secundario: transparente, borde marfil/dorado, texto claro.
- Nunca usar botones planos sin presencia en hero.

### Cards

- Deben tener una funcion visual clara: fotografia, contenido, textura o jerarquia.
- Cards de funciones: imagen real arriba, badge sobre imagen, titulo serif, copy breve.
- Cards de datos internos: mas sobrias, pero con linea dorada o fondo sutil.

### Forms

- Calmados, legibles, con labels claros.
- Fondos oscuros translúcidos o marfil segun contexto.
- Estados de error humanos, no alarmistas.

### App Dashboard

- Menos catedral dramatica que landing, pero debe conservar calidez.
- Usar una tarjeta principal emocional, no un dashboard frio.
- Acciones clave visibles: hablar, orar, escribir.

## 9. Screen Prompts

### Landing Prompt

Disenar una landing web premium para Camino, una plataforma catolica de acompanamiento espiritual y emocional con IA. Debe sentirse como una catedral espanola contemporanea: fotografia real protagonista, luz de vitral viva, azul mariano profundo, oro liturgico, turquesa, bermellon y marfil. Usar composicion editorial con arcos organicos, paneles superpuestos, cards con imagen real y CTA claros. Evitar estetica SaaS generica, iconos vacios y fondos abstractos sin fotografia.

### Dashboard Prompt

Disenar un dashboard calido para Camino, menos corporativo y mas santuario personal. Debe incluir saludo, estado emocional, reflexion diaria, progreso del camino y accesos a hablar, orar y diario. Usar azul profundo, luz dorada, panel principal emocional, textura sutil de vitral y componentes sobrios. No debe parecer un CRM.

### Chat Prompt

Disenar chat IA para acompanamiento espiritual y emocional. Debe sentirse como conversacion en una capilla tranquila: burbujas legibles, modo seleccionado visible, alerta de crisis muy clara y humana, fondo oscuro calido, detalles dorados y mucho aire. No usar apariencia de chatbot tecnico.

## 10. Anti-Patterns

- Plantilla SaaS con cards iguales.
- Logos o iconos vacios sin contenido real.
- Hero sin imagen real.
- Color demasiado apagado para una catedral espanola.
- Exceso de beige que borra la energia espiritual.
- Cruces que no se entienden.
- Texto largo en cards chicas.
- UI religiosa antigua, pesada o culpabilizadora.
