# Design System Camino

La fuente de verdad visual esta en:

```text
.stitch/DESIGN.md
```

## Direccion actual

Camino debe sentirse como una **catedral catolica espanola convertida en producto digital**:

- viva
- luminosa
- premium
- emocional
- catolica sin ser antigua
- bella sin ser pesada
- humana sin ser generica

## Regla principal

La belleza es parte del acompanamiento. Cada pantalla importante debe tener presencia visual real: fotografia, luz, vitral, cruz clara, textura o composicion editorial.

## Para nuevas pantallas

Antes de disenar, revisar:

1. Que imagen real sostiene la pantalla.
2. Donde aparece la cruz o simbolo cristiano claro.
3. Cual es el foco emocional.
4. Que accion principal necesita el usuario.
5. Como evitar que parezca SaaS generico.
