# Fase 5 - Dashboard App

## Objetivo

Crear `/app` como dashboard calido y personalizado con datos reales del perfil.

## Implementado

- Saludo personalizado
- Estado emocional desde `profiles`
- Temas principales desde `profiles.main_topics`
- Reflexion diaria inicial
- Botones: Necesito hablar, Orar ahora, Estoy ansioso/a
- Progreso visual del camino
- Accesos rapidos a Chat, Diario, Oraciones, Reflexiones, Caminos y Comunidad
- Rutas internas base para evitar navegacion rota
- Perfil inicial en `/app/perfil`

## Pendiente

Los modulos internos se completan en sus fases especificas:

- Chat IA: Fase 6
- Diario: Fase 8
- Oraciones: Fase 9
- Reflexiones: Fase 10
- Caminos: Fase 11
- Comunidad: Fase 12

## Como probar

```bash
npm run dev
```

Abrir `/app` con una sesion autenticada.
